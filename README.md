# djmanage

## 介绍
基于django2.2版本和vue的一套完整rbac权限管理系统

## 软件架构
1. 后端语言:django(包括django rest framework)
2. 前端语言:vue


## 安装教程

1.  下载项目
2.  解压项目

## 使用说明

1.  新建数据库,将项目下的sql文件导入建好的数据库
2.  在settings下修改数据库配置
3.  建立项目虚拟环境(可忽略)
4.  启动命令: `python manage.py runserver`

## 其他说明
1. requirements.txt是依赖文件
2. yadmin.sql是数据库文件
